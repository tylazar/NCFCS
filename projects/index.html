STUDENT PROJECTS

Here at New College, we love to combine knowledge with passion to create stellar independent and class projects. Students complete an Independent Study Projects (ISP) each January where they get to choose what to research, study, or create. This freedom of research culminates in a senior thesis in which the student undertakes a year-long project demonstrating their skills in the field. Click on a project below for more information.

Dimensional Doors

Dimensional Doors

This was a project Mei Maddox and William Pembleton worked on in Fall 2019, inspired by a Minecraft mod called Dimensional Doors. In the game, we built a few different levels each exploring an interesting idea.

Our game was built in Unreal 4.23 for the Oculus Rift. Although untested, we believe it is also compatible for play on the HTC Vive but we haven't tested it. This project was built on Windows, therefore we are unsure if the binaries would need to be recompiled if run on a different operating system.

Convo

Convo

Thesis of Ozan Gokdemir, graduated Spring 2019. This project created an Android app allowing New College students to pin their location on a map in order to facilitate face-to-face conversations.

Remindful

Remindful

This project is a prototype created by Skylar Lingenfelser and Ty Lazarchik. It is a phone app that is designed to remind the user when their produce is about to spoil. It also includes reminders for other areas of life such as buying pet food, exercising, and doing laundry.
